# Shyan Hafeez Khan — Engineering Portfolio Website

A single-page portfolio and CV website showcasing my mechanical engineering projects, with a focus on automotive and high-performance engineering. It presents six featured design, simulation and build projects alongside a full CV, and is built to be shared freely with employers.

**Live site:** hosted on GitHub Pages.

## About

I'm a final-year Mechanical Engineering student at Lancaster University. This site brings together the projects I'm most proud of, each with its design imagery, a short overview and the detail behind my contribution, plus my experience, education and skills.

## Features

- Six featured projects with image galleries and click-to-enlarge lightbox
- An "Other Relevant Project Experience" section for additional work
- Full CV laid out on the page, with a downloadable PDF
- Downloadable project portfolio (PDF)
- Responsive layout with a mobile menu
- Motorsport-inspired theme in Petronas teal

## Built with

Plain HTML, CSS and JavaScript, with no frameworks or build step. The entire site is self-contained, so it runs from any static host.

## Structure

```
index.html                      The whole page (markup, styles and script)
assets/
  Shyan-Hafeez-Khan-CV.pdf      Downloadable CV
  Shyan-Khan-Project-Portfolio.pdf  Downloadable portfolio
  projects/                     Project images
```

## Viewing locally

Download the repository and open `index.html` in any web browser. No server or installation is required.
